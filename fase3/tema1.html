Fundamentos de CSS

Primero definimos que es CSS sus siglas en inglés significan Cascading Style Sheets y en español hojas de estilo en cascada, es un lenguaje de marcas que sirven para definir el diseño y aspecto final de un sitio web programado en HTML. Con CSS se le pueden dar formas, colores, estilos, apariencia a la pagina Web bajouna serie de etiquetas, un ejemplo es la etiqueta y etiquetas anidadas que nos ayudaran a esta labor. La estructura en la línea de codigo en CSS esta compuesta de la siguiente forma:

El selector: que nos muestra el tipo de etiqueta que se esta usando.
La propiedad: no indica que el tipo de diseño que se quiere mostrar en la pagina web.
La declaracion: es el codigo como tal, que nos muestra que es lo que se le esta pidiendo a html que realice.
El valor: un ejemplo es en el caso del color, cada color esta representado por un valor, al elegir el valor correcto le podemos indicar mas facilmente a CSS que es lo que queremos.
